IMRSND Recorder Example

This example demonstrates the functionallity for the audio recorder.
It is a command line program showing a menu with some options to select.
The functionallities which this example demonstrates are.

Record audio.
Play the recorded audio.
Store the recorded audio in a wave audio file.

The program should be already compiled inside the 'bin' folder.
If you want to recompile it, just type:

make

To run the 'prog.exe' program, you must execute
the 'run.sh' script if you are in a Linux like shell,
or 'run.bat' if you are under a Windows command line shell.

Linux:
./run.sh

Windows:
run
